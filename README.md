# 📝 To-Do List in Vue 3

As first project in Angular (17+) I decided to build a a simple and modern To-Do List to practice and master the fundamentals of the framework learnt in a Udemy course.

---

## 🚀 Technologies

- ✅ Angular 20
- ⚡ Angular CLI
- 🎨 TailwindCSS
- 💾 localStorage
- ✨ TypeScript
